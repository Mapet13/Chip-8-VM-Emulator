# Chip-8-VM-Emulator
**Chip-8** VM Emulator developed in *Rust-lang* for learning purposes.

#### Debug Mode:
[<img src="https://i.imgur.com/t5xkHKs.png">]()
###### in this mode, you run next instruction by pressing `Space Bar`

#### Keyboard Bindings

|ORIGINAL|  THIS VM  |
|:--------:|:------:|
|  `123C`  | `1234` |
|  `456D`  | `QWER` |
|  `789E`  | `ASDF` |
|  `A0BF`  | `ZXCV` |


#### Usage:
```
cargo run [--release] -- -f=roms/PONG
```
